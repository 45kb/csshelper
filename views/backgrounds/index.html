Backgrounds

{{config.appName}} includes some default backgrounds that can be used on any element.

{{config.appName}} is not intended to style your site but to help you styling it.
If you need more backgrounds you could just create your custom classes, an example for a violet background could be .bg-violet { background-color : violet; }
