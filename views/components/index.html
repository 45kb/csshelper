Components

{{config.appName}} includes some classes that can be used to create components on the fly.
Here is a list of some that will be more useful to you.
{{config.appName}} components can be created simply adding new custom classes to the existent ones.
Usually, when you create a modal or an overlay element, you need some Javascript to interact with it, for example hiding/showing the modal is a JS task.

Overlays

An overlay is an element that covers all your content by default.
It can be very usefull for example, in case that you need to create a modal, or an overlay menu for the user.
