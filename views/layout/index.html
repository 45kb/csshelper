Layout

{{config.appName}} adopts a very simple grid system and logic, .line defines an html row, and .col* defines a column for the .line row

{{config.appName}} does not use any fixed width, if you need to wrap a piece of your html into a fixed width element you could just create a class for it, an example could be .col-fixed-1 { width:900px; }

Compressor

A compressor .line-compress is a .line that is compressed to fit some padding from the parent element left and right edges, the default padding both on left and right sides is 2%, you can easly change it if you need a different setup
.line-compress

Offset

You can use .offset-right* or .offset-left* from 1 up to 10 sizes to offset elements ( min offset is 1 = 10%, max offset is 10 = 100% )
