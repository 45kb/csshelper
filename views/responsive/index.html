Responsive

{{config.appName}} uses media queries to help you define classes and therefore responsive to apply rules based on the user's device.
Just for example, if you want your element to appear only for tablet devices you just have to append .tablet class to that element .

{{config.appName}} responsive classes can be easily extended .
Just, for example, in case that you need to have a background to change on mobile devices, just add a new class to the main mobile media query that you will find in the main source file.

Basics

/*desktop*/ @media screen and (min-width:800px){ /*put in here all desktop and large screen devices rules*/ } /*tablet*/ @media screen and (max-width:800px){ /*put in here all tablet rules*/ } /*mobile*/ @media screen and (max-width:400px){ /*put in here all mobile rules*/ }
* As you can see from the code above, {{config.appName}} defines 3 important media queries, one for desktop, one for tablet and one for mobile devices. These are quite standard responsive widths, but you can easily change the media queries rules, adding your own widths, as well as adding more media queries.

Layout

/*desktop*/ @media screen and (min-width:800px){ } /*tablet*/ @media screen and (max-width:800px){ .line-tablet{ width:100%; margin:0 auto; } } /*mobile*/ @media screen and (max-width:400px){ .line-mobile{ width:100%; margin:0 auto; } }
* As you can see from the code above, {{config.appName}} defines 2 important rules for tablet and mobile devices. The .line-tablet if appended to an element such as a .col* element, makes it fits the full tablet screen width. Let's do an example.

Hiding/Showing

/*desktop*/ @media screen and (min-width:800px){ .mobile, .tablet{ display: none; } .desktop{ display: inherit; } } /*tablet*/ @media screen and (max-width:800px){ .mobile, .desktop{ display: none; } .tablet{ display: inherit; } } /*mobile*/ @media screen and (max-width:400px){ .tablet, .desktop{ display: none; } .mobile{ display:inherit; } }
* As you can see from the code above, {{config.appName}} defines important rules for hiding and showing elements across page resizing.

.desktop

If appened to element it shows the element only for screens that fits the desktop width (default is >= 801px)

.tablet

If appened to element it shows the element only for screens that fits the tablet width (default is <= 800px)

.mobile

If appened to element it shows the element only for screens that fits the mobile width (default is <= 400px)
