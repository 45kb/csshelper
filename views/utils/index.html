Utils

{{config.appName}} includes some classes that can be used to do simple tasks like for example .uppercase which returns your text to UPPERCASE
{{config.appName}} utils classes can be easily extended .
Just, for example, in case that you need to have a border-radius only on left side of the elmeent you could create a nice .radius-left-3 { border-radius:3px 0px 0px 3px; }
